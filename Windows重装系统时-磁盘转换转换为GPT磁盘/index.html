Windows重装系统时,磁盘转换转换为GPT磁盘

发表于 2017-09-09 | | 浏览 次

  • 问题

    Windows重装系统时,弹窗提示
    Windows无法安装到这个磁盘，选中的磁盘具有MBR分区表。在EFI系统上，Windows只能安装到GPT磁盘

  • 解决

    在磁盘分区的界面下(事实上,我试过不止这个界面可以,其他界面也许可以,可以自己试试),shirt+F10打开命令行依次键入(#后面为注释)

回到磁盘分区界面刷新,稍等片刻,之后就可以正常选择系统安装的分区了.
